## 一、xAI 发布 Grok Build 早测版——AI 编程 CLI 工具【⭐⭐⭐】
**来源：** xAI News
**链接：** https://x.ai/news/grok-build-cli
**关键词：** Grok Build, xAI, CLI, 编程Agent, SuperGrok Heavy, 子代理, MCP

### 摘要

xAI 在 2026 年 5 月 14 日发布了 **Grok Build** 的早期 Beta 版本——一个面向专业软件工程场景的编程 Agent 和 CLI 工具。这是 xAI 正式杀入 AI 编程助手赛道的信号，直接对标 Claude Code、Cursor、GitHub Copilot 等产品。

核心功能：

1. **Plan → Review → Approve 工作流**：复杂任务可以先让 Grok Build 生成计划，用户可以审批、评论修改或重写计划，然后才执行。每次代码变更都会展示清晰的 diff，方便审查。

2. **无缝兼容现有配置**：项目中已有的 AGENTS.md、plugins、hooks、skills 和 MCP 服务器都能直接用，开箱即用。

3. **并行子代理**：大任务可以拆成多个专业子代理并行干活，支持 worktree 集成，子代理可以在独立 worktree 中运行。

4. **Headless 模式（-p）**：适合集成到脚本和自动化流水线中。还提供完整的 ACP 支持，可以用来构建自己的 bot 和 Agent 编排应用。

目前只面向 **SuperGrok Heavy** 订阅者开放。安装方式也很简单：`curl -fsSL https://x.ai/cli/install.sh | bash`。

**影响分析**：xAI 之前主要是做对话式 Grok，现在推代码 Agent 意味着他们要把 Grok 变成一个真正的开发工具。SuperGrok Heavy 这个高级订阅层级（比标准 SuperGrok 更高端）表明定位在专业开发者市场。和 Anthropic 的 Claude Code、OpenAI 的 Codex CLI 形成直接竞争——AI 编程工具的三国杀格局更加清晰了。
