# 开源项目

## 一、Anthropic开源skills——用自然语言写AI Agent工具【⭐⭐⭐】
**来源：** GitHub Trending Daily
**链接：** https://github.com/anthropics/skills
**关键词：** Anthropic、skills、MCP、Agent工具、自然语言编程

### 摘要
Anthropic开源了skills工具库，这是一套让AI Agent可以**通过自然语言描述来自动生成和注册工具**的框架。和传统的function calling不同，用户只需要用话描述一个技能（比如"帮我监控仓库库存水位，低于阈值时发Slack通知"），系统就能自动生成对应的MCP工具并注册使用。

这实际上是Anthropic MCP生态的关键一环——以前你需要写代码定义工具、注册工具、连接到Agent。现在理论上只需要用自然语言描述就够了。对于非开发者和快速原型来说，这个门槛降低非常明显。

## 二、n8n-mcp——用Claude/Cursor自动构建n8n工作流【⭐⭐】
**来源：** GitHub Trending Daily
**链接：** https://github.com/czlonkowski/n8n-mcp
**关键词：** n8n、MCP、工作流自动化、Claude Code、Windsurf、Cursor

### 摘要
n8n-mcp是一个MCP服务器，它让Claude Desktop、Claude Code、Windsurf、Cursor等AI编程工具可以直接**通过对话来构建n8n工作流**。也就是说，你不再需要手动拖拽n8n的节点去配置自动化流程，只需要跟AI说"帮我搭一个当GitHub有新Issue时发送钉钉通知的工作流"，AI就会自动在n8n中创建完成。

这个项目的价值在于，它把工作流自动化的两个热门方向（n8n + AI编码代理）打通了——AI不仅能帮你写代码，还能帮你搭自动化。

## 三、qiaomu-anything-to-notebooklm——把任何内容转成NotebookLM格式【⭐⭐】
**来源：** GitHub Trending Daily
**链接：** https://github.com/joeseesun/qiaomu-anything-to-notebooklm
**关键词：** NotebookLM、Claude Skill、内容转换、多源输入

### 摘要
这是一个Claude Skill（MCP工具），支持将多种来源的内容（微信公众号文章、网页、YouTube视频、PDF、Markdown、甚至搜索查询）自动处理成NotebookLM可用的格式，并能转换为播客、PPT、思维导图、测验等不同输出形式。

对于NotebookLM用户来说，这是个非常实用的工具。原来你需要手动把各种内容复制粘贴进NotebookLM，现在通过一个Claude Skill就可以一键处理。
