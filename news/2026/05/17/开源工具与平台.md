## 一、oven-sh/bun【⭐】
**来源：** GitHub Trending Daily
**链接：** https://github.com/oven-sh/bun
**关键词**：JavaScript运行时，高速，全合一
### 摘要
Bun——极快的JavaScript运行时、打包器、测试运行器和包管理器。已经不是新项目了，但持续在Trending上，说明项目还在快速增长。相比Node.js和Deno，Bun的开发体验和性能有明显优势。

## 九、pranshuparmar/witr【⭐】
**来源：** GitHub Trending Daily
**链接：** https://github.com/pranshuparmar/witr
**关键词**：进程溯源，诊断，Go
### 摘要
"Why Is This Running"的缩写——一个轻量级进程溯源诊断工具。今天也在Trending上（连续第二天），如果你用的是Mac/Linux，遇到"电脑上这个进程是干嘛的"的困惑时，witr能直接给出答案。用Go语言写的，安装简单。
