# AI应用与模型

## 一、dograh-hq/dograh【⭐⭐】
**来源：** GitHub Trending Daily
**链接：** https://github.com/dograh-hq/dograh
**关键词**：开源语音Agent，语音平台，Voice Agent
### 摘要
开源的语音Agent平台，可以自托管。语音Agent是目前AI应用最活跃的赛道之一，从简单语音助手到复杂的语音交互系统需求都在暴涨。开源方案给了开发者更多控制权，也更适合隐私敏感场景。

## 二、Light-Heart-Labs/DreamServer【⭐⭐】
**来源：** GitHub Trending Daily
**链接：** https://github.com/Light-Heart-Labs/DreamServer
**关键词**：本地AI，一站式，LLM推理，RAG，图像生成
### 摘要
"本地AI无处不在"的一站式方案——涵盖LLM推理、聊天界面、语音、Agent、工作流、RAG、图像生成。口号是"不需要云，不需要订阅"。对标Ollama、LM Studio等本地AI方案，但整合得更全面。说明本地AI方案正从"跑个模型"进化到"跑一整个AI工作站"。

**来源：** GitHub Trending Daily
**链接：** https://github.com/BigBodyCobain/Shadowbroker
**关键词**：OSINT，开源情报，飞行追踪，卫星追踪，AI分析
