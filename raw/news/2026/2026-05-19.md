### Anthropic acquired Stainless ⭐⭐⭐
**来源：** Anthropic News
**链接：** https://www.anthropic.com/news/anthropic-acquires-stainless
**关键词：** Anthropic, Stainless, SDK, MCP, Agent连接

### 摘要

AI 的边界正在从回答问题转向自主行动——而 Agent 的能力取决于它能连接多少系统。Anthropic 收购了 SDK 与 MCP 工具公司 Stainless（成立于 2022 年），后者从 Anthropic API 诞生起就在生成其官方 SDK。数以百计的公司靠 Stainless 生成 SDK、CLI 和 MCP 服务器，覆盖 TypeScript、Python、Go、Java、Kotlin 等多种语言。创始人 Alex Rattray 表示 Stainless 团队将继续做自己热爱的工作。这次收购明确释放了一个信号：Anthropic 在押注 Agent 生态的基础设施层。
