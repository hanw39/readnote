# AI

## 一、Anthropic Claude Code v2.1.116 发布【⭐】
**来源：** Claude Code Changelog
**链接：** https://github.com/anthropics/claude-code/blob/main/CHANGELOG.md#21116
**关键词：** Claude Code、Anthropic、发版、开发者工具

### 摘要

Claude Code v2.1.116 发布，主要是一系列 Bug 修复和体验优化：

**修复类：**
- 修复了希伯来语、阿拉伯语等 RTL（从右到左）文本在 Windows Terminal、Console 和 VS Code 集成终端中渲染不正确的问题
- 修复了 LSP 服务器因文件 URI 格式错误在 Windows 上无法工作的问题
- 修复了 `--plugin-dir` 参数现在优先使用本地开发副本而非同名已安装 marketplace 插件
- 修复了"无标题"会话删除按钮失效的问题
- 改进了集成终端中鼠标滚轮的响应灵敏度，支持 terminal-aware 加速
- VSCode：修复了在 Windows 上从 VS Code 集成终端启动时终端不打开的问题

**总体来说，** v2.1.116 是一个维护性版本，重点修复了 Windows 平台上的多个遗留问题，包括 RTL 语言渲染、LSP 兼容性和终端交互体验。如果你主要在 Windows 上使用 Claude Code，这个版本值得更新。

## 二、Anthropic Claude Code Hackathon 获奖项目公布【⭐】
**来源：** Claude Blog
**链接：** https://claude.com/blog/meet-the-winners-of-our-built-with-opus-4-6-claude-code-hackathon
**关键词：** Claude Code、hackathon、Opus 4.6、Anthropic

### 摘要

Anthropic 公布了"Built with Opus 4.6"Claude Code Hackathon 的获奖名单，这是围绕 Claude Opus 4.6 模型和 Claude Code 工具举办的开发者竞赛。官方博客展示了各个获奖项目的详细信息，涵盖了不同应用场景的创新解决方案。

**对开发者的参考价值。** 获奖项目通常代表当前 Claude Code 最前沿的用法和最佳实践，对于想了解如何用好 Claude Code 做 AI 原生开发的开发者来说，这些案例是很好的学习素材。
