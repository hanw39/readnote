## Academic Research Skills for Claude Code v3.9.3【⭐⭐】
**来源：** GitHub Trending Daily
**链接：** https://github.com/Imbad0202/academic-research-skills
**关键词：** Claude Code, 学术研究, AI辅助写作, 论文管线

一套完整的 Claude Code 技能套件，覆盖学术研究的完整管线——从文献调研到论文发表。核心设计哲学是 human-in-the-loop：AI 负责脏活累活（查引用、格式化、逻辑一致性检查），人负责真正需要脑子的部分（定义问题、选择方法、解读结果、写论点）。

v3.9.3 引入了几个新特性：Socratic dialogue 引导论文结构规划、Style Calibration 从历史写作中学习个人风格、Writing Quality Check 检测机器生成感。引用方面集成了信任链 frontmatter 和三层定位基础设施，对标 Zhao et al.(2026) 对 arXiv 上 250 万篇论文的审计结果——2025 年约有 14.7 万条虚构引用。

## 12-Factor Agents【⭐⭐】
**来源：** GitHub Trending Daily
**链接：** https://github.com/humanlayer/12-factor-agents
**关键词：** AI Agent, 架构原则, 生产部署, 上下文工程

受到 12 Factor App 方法论启发，这套原则系统回答了"怎么构建真正能上生产的 LLM 应用"。作者 Dex 遍历了 crewai、langchain、smolagents、langraph、griptape 等框架后得出结论：真正在客户面前跑得好的 AI Agent，大多是自己手撸堆栈的。

核心洞见：大部分自称"AI Agent"的产品其实没那么 Agentic——它们是确定性代码为主，在精确节点掺入 LLM 步骤。好的 Agent 不遵循"给 prompt + 给工具包 + 循环到目标"的模式，而是由常规软件工程构成。项目还提供了一个 `npx/uvx create-12-factor-agent` 脚手架。
