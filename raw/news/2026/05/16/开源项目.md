# 开源项目

## 一、xAI × Nous Research：Grok接入开源Agent Hermes【⭐⭐⭐】
**来源：** xAI News
**链接：** https://x.ai/news/grok-hermes
**关键词：** xAI、Grok、Hermes Agent、Nous Research、开源Agent、Grok 4.3

### 摘要
xAI 和 Nous Research 联手，将 Grok 的能力直接接入开源 Agent 框架 **Hermes Agent**。即日起，拥有 Grok 订阅的用户可以在 Hermes Agent 中使用 Grok 4.3 的文本对话/高级推理能力、Grok Text-to-Speech 语音回复，以及 Grok Imagine 图像/视频生成。

Hermes Agent 是 Nous Research 开发的开源、自改进 Agent，可以持久运行在任何电脑、沙箱或 VPS 上，跨会话维护长期记忆，并能连接 WhatsApp、Discord、Telegram、Signal 等消息平台。安装只需要一行命令：

```bash
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
```

然后通过 `hermes model` 选择 xAI Grok OAuth 认证，登录后即可通过 `hermes --tui` 文字界面使用。

**这件事的意义**：这是 xAI 在模型 API 之外的第一个 Agent 生态动作。不同于 OpenAI 走封闭的 Agent SDK 路线，xAI 选择拥抱开源社区——把 Grok 作为 Hermes Agent 的后端模型提供商。这种"开放生态"策略和 xAI 宣称的"AI for the world"叙事一致，也给了开发者一个不依赖 OpenAI/Anthropic 封闭生态的选择。
