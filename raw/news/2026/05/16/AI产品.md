# AI产品

## 一、xAI开放Grok至第三方Agent平台：Hermes Agent支持Grok 4.3【⭐⭐⭐】
**来源：** xAI News
**链接：** https://x.ai/news/grok-hermes
**关键词：** xAI、Grok 4.3、Hermes Agent、模型订阅、多模态

### 摘要
xAI 宣布 Grok 订阅用户现在可以将 Grok 连接到 Hermes Agent（Nous Research 的开源 Agent 框架）。支持的 Grok 能力包括：Grok 4.3 文本对话/高级推理、Grok Text-to-Speech 语音回复、Grok Imagine 图像/视频生成。

这是 xAI 首次将其模型能力开放给第三方 Agent 平台，走出了"围墙花园"的第一步。所有订阅层级（包括最低档）均可使用这项功能。对用户来说，这意味着可以在一个统一的 Agent 框架中同时获得 Grok 的推理能力、语音输出和生成式图像能力，并通过 Hermes 的长期记忆和多平台消息接入（WhatsApp/Discord/Telegram/Signal）实现持续的 AI 助手体验。

对于 xAI 自身而言，这是在模型能力之外构建生态的起步动作——如果更多开源 Agent 框架跟进支持 Grok，xAI 将在 Agent 基础设施层获得新的分发渠道。
