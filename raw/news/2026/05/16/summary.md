# 今日AI资讯总结

## 一、今天发生了什么（整体趋势）
今天只有一条新闻，但信息量不小：xAI 正式把 Grok 接入开源 Agent 框架 Hermes Agent。这是 xAI 在模型 API 之外的第一个生态布局动作——不同于 OpenAI 的封闭 Agent SDK 路线，xAI 选择拥抱开源社区，把 Grok 4.3、Grok Text-to-Speech 和 Grok Imagine 打包成一个可选后端，供 Hermes Agent 用户使用。

## 二、重点方向
- **AI Agent 生态开放化：** xAI 加入开源 Agent 生态，提供 Grok 作为后端模型选项

## 三、值得重点关注的3件事
1. **xAI 的生态策略转向：** 以前 xAI 的 Grok 只能在 X（原 Twitter）内部使用，现在首次开放给第三方 Agent 平台。这是一个温和的"开放化"信号——xAI 可能在逐步靠近开源社区
2. **Hermes Agent 的定位：** Nous Research 的 Hermes 一直以开源、自改进、跨平台著称。这次接入 Grok，意味着一线大模型（Grok 4.3）和开源 Agent 框架的融合又向前了一步
3. **Agent 生态的分化：** 现在的格局大致是——OpenAI 做封闭 SDK + GPTs，Anthropic 做 MCP 开放协议 + Claude Code，xAI 做模型后端子给开源社区。三种路线的分化越来越清晰

## 四、一句话总结
xAI 迈出生态开放第一步，Grok 走出围墙花园进入开源 Agent 世界。
