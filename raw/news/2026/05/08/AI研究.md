## 一、LLM vs 人类的策略思维差异【⭐⭐】

**来源：** The Batch Issue 351
**链接：** https://www.deeplearning.ai/the-batch/issue-351
**关键词：** 策略思维, LLM研究, 游戏理论, AI行为

### 摘要
德克萨斯大学和 Google 的研究人员通过石头剪刀布游戏揭示了 LLM 和人类策略思维的差异。虽然 LLM 可以表现出类人的行为，但在策略决策方式上明显不同于人类。研究发现 LLM 的决策模式更接近优化算法而非人类直觉，这提醒我们：LLM 看似"像人"，但底层机制完全不同。

## 二、Prompting 方式的演变【⭐】

**来源：** The Batch Issue 351（Andrew Ng 的公开信）
**链接：** https://www.deeplearning.ai/the-batch/issue-351
**关键词：** Prompting, AI教育, Andrew Ng

### 摘要
Andrew Ng 在最新的公开信中对比了 2022 年 ChatGPT 刚出来时和现在的 Prompting 方式差异。现在大家已经不是简单提问题了，模型可以做深度研究（花几分钟思考）、吞入海量文档作为上下文、使用网络搜索和其他工具。他为此推出了一门新课程《AI Prompting for Everyone》，面向所有水平的学习者。

## 三、关于 Coding Agent 的四种工作类型加速程度【⭐⭐】

**来源：** The Batch Issue 350（Andrew Ng 的信）
**链接：** https://www.deeplearning.ai/the-batch/issue-350
**关键词：** Coding Agent, 软件开发, 效率

### 摘要
Andrew Ng 将软件工作分为四类，按 Coding Agent 加速程度排序：

1. **前端开发**（加速最多）：TypeScript/JavaScript/React 是模型最擅长的领域，还能通过浏览器检查自己写的页面迭代修改
2. **后端开发**（加速中等）：更容易产生隐蔽 bug 和安全问题，数据库迁移需要谨慎处理
3. **基础设施**（加速较少）：复杂网络配置、可靠性工程等对模型来说知识还很薄弱
4. **研究**（加速最少）：研究不仅是写代码，还有假设提出、实验设计和结果解读

Ng 强调现在他已经让前端团队实现产品的速度比一年前快得多，但对研究团队的预期没有太大变化——这对如何组织软件团队有重要启发。
